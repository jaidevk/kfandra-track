The basic points system in place as of now is;  
Attendance:  
1. Confirmation: 100 each as per attendance confirmation (if there are 8 players that have attended and you informed 1st then 800, 2nd then 700.....so on)  
2. Sequence of coming to the ground of those 8 players...1st to arrive 800, 2nd to arrive 700....so on.  
3. Confirming for availability beyond 7:30am on Tuesday and Thursday is 500 points bonus  
  
Packing:  
1. GWW unpacking: 500  
2. GWW packing: 500  
3. Session packing: 200  
  
Fooba/Full Game  
1. Win 1000  
2. Draw 100  
3. On Big Goal  
     a. Goal 500  
     b. Assist 200  
     c. Pre-Assist 100  
     d. Save 500  
     e. Goal Conceded (200)  
4. On Rebound Wall  
     a. Goal 200  
     b. Assist 100  
     c. Pre-Assist 50  
     d. Save 0  
     e. Goal Conceded 0  
5. Goal Line Clearance 500  
6. Switch 100  
  
KLCSGR / Short Game  
1. Win 1000  
2. Draw 100  
3. All else Same as Fooba/Full Game if goal scoring on rebound wall or is difficult as per Sir's discretion  
Otherwise  
a. Goal 200  
b. Assist 100  
c. Pre-Assist 50  
d. Goal Line Clearance 200  
  
End Game Drill & Progression  
1. Completing each pass 200  
2. Completing set of passes as given by Sir 500  
3. Each time a defender blocks or successful tackle 500 (each defender)  
4. Win 1000  
  
3 and in / 4 and in Drill etc.  
Same as Fooba/Full Game points.
